ThreadWeaver

AI chat where every conversation is versioned. Branch from any message, explore alternatives, paste images, save highlights to your notebook.

View on GitHub Powered by AgentStateGraph

v0.3.0-beta · Open source · MIT/Apache-2.0

⑂

Branch Any Message

Click any message to fork the conversation. Explore "what if I'd asked differently?" without losing the original thread. Visual branch tree shows your conversation history.

🖼

Multimodal

Paste images, drop files. Works with Claude, GPT, Gemini, Grok, and vision-capable local models.

🔧

Tool Calling + MCP

Built-in file tools plus any MCP server. The AI reads files, runs commands, queries databases — acts, not just talks. Works with Ollama too.

📓

Notebook

Save any message to your notebook with a tag. Add comments, branch new chats from highlights, copy to clipboard.

📁

Projects

Group related conversations into projects with shared context. Like ChatGPT projects, but open source and versioned.

📊

Full Provenance

Every message exchange is a versioned commit with intent, reasoning, and confidence. View the complete timeline via AgentStateGraph.

🎨

Customizable

10 accent colors, 10 background themes. Make it feel like yours. Settings persist across sessions.

🤖

6 LLM Providers

Claude, GPT, Gemini, Grok, Ollama, or any OpenAI-compatible endpoint. Bring your own keys. Switch providers per conversation.

🚀

Self-Hosted

Docker, native install, or PicoClaw/Jetson ARM64. Your data stays yours. No accounts, no tracking.

Get Started in 60 Seconds

Docker

git clone https://github.com/nosqltips/ThreadWeaver
cd ThreadWeaver
docker compose up -d
# → http://localhost:3000

With Local Ollama

docker compose -f docker-compose.yml \
  -f docker-compose.ollama.yml up -d
docker compose exec ollama \
  ollama pull llama3.2:3b

Native (PicoClaw/Linux)

git clone https://github.com/nosqltips/ThreadWeaver
cd ThreadWeaver
bash install-native.sh
